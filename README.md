# Projetos-Python

🐍 Exercícios de Fundamentos de Python – SENAI

Este repositório reúne exercícios desenvolvidos durante o curso de Fundamentos de Python do SENAI, com foco no aprendizado prático da linguagem e no desenvolvimento do raciocínio lógico.

Os códigos apresentados abordam desde conceitos básicos até estruturas fundamentais da programação, servindo como base sólida para evolução na área de desenvolvimento de software.

🚀 Conteúdos Abordados
Variáveis e tipos de dados
Operadores lógicos e matemáticos
Estruturas condicionais (if, else, elif)
Estruturas de repetição (for, while)
Listas, tuplas e dicionários
Funções
Entrada e saída de dados
Lógica de programação
🎯 Objetivo

O objetivo deste repositório é consolidar os conhecimentos adquiridos no curso, além de servir como material de consulta e prática contínua na linguagem Python.

💡 Observações

Os exercícios foram desenvolvidos com foco educacional, priorizando clareza, simplicidade e entendimento da lógica, sendo ideais para iniciantes na programação.
